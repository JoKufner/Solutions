Vorgehen zum Importieren der Lösung:

1. Sharepoint-Listen in eine Sharpoint-Seite mittels CSV importieren
2. Umgebungsvariablen setzen
2.1 SharePointPageURL: Die URL der gewählten SP-Seite
2.2 SPList_Urlaub_ID: ID der Liste Urlaub
2.3 SPList_Urlaubsplanung: ID der Liste Urlaubspolanung
3. Datenquelle in der Canvas App aktualisieren (neu beziehen. Vorher bestehende Datenquelle "Urlaubsplanung" aus der App löschen, damit der Name passt).
